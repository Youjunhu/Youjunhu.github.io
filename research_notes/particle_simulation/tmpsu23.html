5.8 Evaluation of particle number density

The particle number at the boundary grid-points j = 0 and j = N needs special treatment. The above treatment do not include the contribution to n0 from the left cell of the j = 0. Since we use periodic boundary condition, the contribution to n0 from the left cell of the j = 0 grid is identical to the contribution to nN from the left cell of the j = N grid. The latter has already been calculated in the above, which can be added to n0 obtained above to get the ﬁnal n0. The same situation applies for nN. After these treatment, we have n0 = nN.

Dividing the particle number nj by the cell size Δ gives the number density.
